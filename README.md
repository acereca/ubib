Universal Bibliography for BibTeX
=================================

This bibfile should contain all of my read papers and books for easy referencing in any LaTeX file.


## Templates

### Article
@article{uniquename,
    title={},
    author={Name, Full},
    journal={},
    pages={},
    year={}
}

### Website
@oneline{uniquename,
    author={},
    title={},
    url={},
    urldate={YYYY-MM-DD},
    note={}
}
